# ✨ Professional Weather Dashboard

A beautiful app-like Weather Application built using **HTML, CSS, and JavaScript** with login functionality and live weather data using Weather API.

# 🚀 Features

✅ Login Screen

✅ Search Weather by City

# ⚡ How to Run the Project

## 1️⃣ Clone Repository

```bash
git clone https://github.com/gayatri951/weather-app.git
```

---

## 2️⃣ Open Project Folder

```bash
cd weather-app
```

---

## 3️⃣ Run Website

Open:

```bash
index.html
```

using Live Server in VS Code.

---

# 🔑 Setup API Key

Inside:

```bash
app.js
```

Replace:

```javascript
const apiKey = "YOUR_API_KEY";
```

with your actual API key.
